BLIP propose multimodal mixture of encoder-decoder, which is a unified V-L model. Featured in:

  • Unimodal encoder is trained with ITC loss
    • to align the feature space of visual transformer and language transformer
    • a momentum encoder is introduced to produce features and soft labels
  • Image-grounded text encoder is trained with ITM loss
    • use cross-attention(CA) layers to model V-L interactions
    • adopt hard negative mining strategy
  • Image-grounded text decoder is trained with Cross Entropy loss
    • use causal self-attention layers, and share same CA layers and FFN as encoder
  • CapFilt as data augmentater

BLIP-2 introduce Q-Former as a trainable module to bridge the gap between a frozen image encoder and a frozen LM.

  • learnable Q-Former is initialized form pre-trained BERT, having 188M parameters
  • image feature is extracted via ViT, and fed into cross-attention layers inside Q-Former
  • joint optimize 3 pre-training objectives that share same input format and model parameters
    • ITC perform uni-modal SA mask
    • ITM perform bi-directional SA mask
    • ITG perform multi-modal causal SA mask
      • enforce queries to extract visual features most relevant to text
  • Followed by FFC layer to linearly project the output query embeddings into same dimension as text embedding, so that projected query embeddings are prepended to input text embeddings to condition LLM to generate text.

InstructBLIP introduce an instruction-aware Q-Former to enable flexible visual feature extraction according to given instructions.

  • transform 26 datasets into instruction-response format, which are grouped into 11 task categories.
  • instructions are fed into both frozen LLM and Q-Former
