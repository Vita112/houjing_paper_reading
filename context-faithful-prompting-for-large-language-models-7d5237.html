Context-faithful Prompting for Large Language Models

Posted on Fri, Nov 3, 2023

1. Introduction

  • LLMs achieved remarkable progress on knowledge-driven tasks such as question answering and commonsense reasoning
    • LLMs encode parametric knowledge about open-world facts, but it may be misconceptions or obsolete facts.
    • overly rely on it lead to overlook knowledge in context, resulting in unsatisfied results.
  • this paper focus on faithfulness of LLMs, so LLM should concern validity and reliability of its extraction process.
    • given task of to extract decision-related info, LLM should induce what is described in the context. Besides, it should selectively give up prediction when no known info is given in the context.
    • concentrate on 2 sub-problems and propose 2 method to solve them
      • entity-based knowledge conflict
      • prediction with abstention
      • effective prompts
        • reformulate context and answers to opinion-based QA
      • appropriate in-context demonstration
        • counterfactual demonstrations

    2. Method

  • input: (c,q)(c, q)(c,q) or (c,q,options)(c, q, options)(c,q,options); output: free-form text or a choice
  • only change the prompts without finetuning

2.2 Counterfactual demonstration

start with a labelled set of counterfactual instances and a test instance and then use KATE (Liu et al., 2022) to retrieve the most relevant counterfactual instances as demonstrations.

3 Experiments

  • model :InstructGPT model (text-davinci-003, 175B parameters) and LLama-2-7B-chat
  • baseline: use base prompt

3.1 using counterfactual datasets to evaluate the knowledge conflict

two datasets: natural queations from MRC and Re-TACRED for relation extraction

modify context to support a counterfactual answer to create counterfactuals

  • for MRC, replace the gold entity answer in the context with a randomly sampled entity of same entity from same corpus
  • for RE, randomly sample a context that has the entity mentions of the same type but having diff relations, then insert the original entities into the sampled context

a faithful LLM should update its prediction to the new answer.
